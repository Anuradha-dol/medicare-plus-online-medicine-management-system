Online Medicine Management System - Website Features

Public Website
-  landing page with pharmacy theme.
- Login page.
- Sign-up page.
- Sign-up supports two roles:
  - User
  - Pharmacist
- Pharmacist sign-up asks extra details:
  - Pharmacy Name
  - Pharmacy Address

Role System
- Three roles:
  - Admin
  - Pharmacist
  - User
- Each role has a separate dashboard and separate access.
- Users cannot access pharmacist/admin pages.
- Pharmacists cannot access admin/user-only pages.
- Admin controls management areas.

Admin Features
- Admin dashboard with analytics.
- View users.
- Delete users.
- View pharmacists.
- Approve pending pharmacists.
- Delete pharmacists.
- View all orders.
- View order status and transport method.
- View customer order messages/issues.
- Reply to customer messages.
- Change message status:
  - Open
  - In Review
  - Resolved
  - Closed

Pharmacist Features
- Pharmacist account waits for admin approval.
- Pending pharmacist cannot login.
- Approved pharmacist can login.
- Add medicines.
- Upload medicine image.
- Update own medicines.
- Delete own medicines.
- Select transport methods available for medicines:
  - Standard Medical Courier
  - Express Medical Courier
  - MediCare Delivery Service
- View orders for their medicines only.
- Change order status:
  - Pending
  - Approved
  - Completed
  - Cancelled
- When setting Approved, pharmacist enters expected delivery date/time.
- Approved shows to user as On The Way.
- Completed means order delivered/reached user.

User Features
- User can register and login without admin approval.
- Browse available medicines.
- See medicine images.
- See medicine price, stock, category, description.
- Add medicine to cart.
- Cart shows medicine image and live subtotal.
- Quantity change updates price total.
- Place order.
- User can only select transport methods enabled by pharmacist.
- Disabled transport methods are visible but faded and cannot be selected.
- View orders in clear status sections:
  - All
  - Pending
  - On The Way
  - Completed
  - Cancelled
- User can send message/problem to admin for an order:
  - Delivery Problem
  - Order Not Received
  - Order Received
  - Medicine Problem
  - Other
- User can see admin response under the order.

Order Flow
- User places order.
- Order starts as Pending.
- Pharmacist approves order and adds expected delivery date/time.
- User sees order under On The Way.
- Pharmacist marks Completed when delivered.
- User sees order under Completed.
- Cancelled orders go to Cancelled section.

UI Features
- Advanced colorful pharmacy UI theme.
- Medicine images in cards, cart, and orders.
- Professional order cards/receipt style.
- Beautiful popup/toast messages for login, register, cart, order, medicine, approval, and message actions.
- Responsive layout for desktop and mobile.
- Clean status badges and dashboards.
